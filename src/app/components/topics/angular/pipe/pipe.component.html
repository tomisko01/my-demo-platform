How to use | pipes

Template mechanism to transform data and display it in another form to the user

Date pipes

Result:

startDate value = {{ startDate }}
startDate value with date pipe = {{ startDate | date }}
startDate value with date pipe formatted dd/MM/yy = {{ startDate | date: 'dd/MM/yy' }}

Code:

  
    {{ datePipeCode }}

String pipes

Result:

title value = {{ title }}
title value with lowercase = {{ title| lowercase }}
title value with uppercase = {{ title| uppercase }}

Code:

  
    {{ stringPipeCode }}

Number pipes

Result:

number value = {{ numberValue }}
number value with digits info = {{ numberValue | number: '3.3-5' }}
number value as currency = {{ numberValue | currency }}
number value as currency = {{ numberValue | currency: 'EUR' }}

Code:

  
    {{ numberPipeCode }}

Percent pipes

Result:

rateValue = {{ rateValue }}
rateValue as percent {{ rateValue | percent }}

Code:

  
    {{ percentPipeCode }}
