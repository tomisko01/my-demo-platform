How to use View Child

Used to get reference to child components

Get reference to child component

We will receive reference to component

Child component received by @ViewChild 
{{ child | json }}

If we have more child components, we will receive reference to first one

Template reference

Child component received by @ViewChild with template reference 
{{ childByTemplateRef | json }}
 Another child component 
{{ otherChildByTemplateRef | json }}

Element reference

We will receive reference to DOM element

Child component received by @ViewChild with element reference after view checked 
{{ childElementRef | json }}
 Here we receive a div element

Get component as element reference

We will receive reference to DOM element

Child component received by @ViewChild with element reference after view checked 
{{ childComponentAsElementRef | json }}
 Here we receive a child element as plain html element

View child cannot query in child component! We can only get children components on parent level, children of children will not be received
