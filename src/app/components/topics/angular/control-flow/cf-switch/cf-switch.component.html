How to use @switch in v17

Switch usage by property (category):

Given object:

      
{{object1 | json}}

Result:

@switch(object1.category){ @case("BEGINNER"){
BEGINNER
} @case("INTERMEDIATE"){
INTERMEDIATE
} @case("ADVANCED") {
ADVANCED
} }

Code:

      
{{switchCode}}

Switch usage with list and default option

Given list:

      
{{listToShow | json}}

Result:

@for(item of listToShow; track item.name){

Object name: {{item.name}} @switch(item.category){ @case("BEGINNER"){

BEGINNER
} @case("INTERMEDIATE"){
INTERMEDIATE
} @case("ADVANCED") {
ADVANCED
} @default {
UNKNOW
} } }

Code:

      
{{switchListCode}}
